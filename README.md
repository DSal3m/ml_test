# ml_test

Базовый шаблон репозитория для ML/DS экспериментов.

## Структура

- `src/` — исходный код
- `data/` — локальные данные (в `.gitignore`)
- `notebooks/` — ноутбуки
- `tests/` — автотесты

## Быстрый старт

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
pytest
```
